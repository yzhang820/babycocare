# 小宝同护 MVP

这是“小宝同护”的第一版可交互原型，重点验证核心照护流程：

- 宝宝档案：昵称、出生日期、当前省市、喂养方式、出生体重、自动目标奶量。
- 宝宝头像：支持上传宝宝照片作为首页头像。
- 省市选择：基于当前城市展示天气、温度、紫外线、风力和外出准备建议。
- 疫苗提醒：在首页提示近期必打和可选自费疫苗，避免错过预约窗口。
- 自动奶量：按当前体重自动估算目标奶量；母乳喂养不强行设定每日奶量。
- 今日清单：提醒事项式单列列表，包含一天多次喝奶、尿布、睡眠、陪玩、外出准备和交接。
- 完成填写：点击清单圆点后在卡片内行内调整数值、滑杆和备注，再进入报告时间线。
- 智能调整确认：奶量明显低于计划时，先提醒照护人确认，再调整后续安排。
- 照片视频记录：记录重要画面和成长片段。
- 成长曲线：记录体重、身高，绘制趋势图，并给出是否落在 Demo 常见范围内的提示。
- 报告：根据清单、照片视频记录生成晚间总结、合并时间线，并支持查看历史报告。
- 提醒时间：早上查看清单、晚上查看报告的时间可调整。

## 打开方式

直接用浏览器打开 `index.html` 即可体验。当前版本把数据存在本机浏览器里，点击右上角重置按钮可以恢复可验证示例数据。
